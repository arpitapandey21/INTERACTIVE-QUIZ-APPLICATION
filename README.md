## DESCRIPTION OF THE PROJECT
Arpita Web Mastery is a modern, interactive, and visually engaging web-based quiz application designed to test and strengthen core web development knowledge. This project focuses on fundamental and intermediate concepts of HTML, CSS, JavaScript, React, and modern web technologies. It is built using pure HTML, CSS, and JavaScript without any external frameworks, demonstrating strong frontend development fundamentals.

The application presents 15 carefully curated multiple-choice questions covering essential web development topics such as HTML structure, CSS styling, Flexbox, Grid, JavaScript array methods, ES6 features, JSON, REST APIs, Local Storage, and React concepts like useState and React Router. Each question is timed (30 seconds), making the experience dynamic and engaging while encouraging quick thinking and accuracy.

✨ Key Features

🎯 15 Professional Web Technology Questions

⏳ 30-Second Countdown Timer for Each Question

📊 Real-Time Score Tracking

🔄 Circular Animated Progress Indicator

🎨 Glassmorphism & Gradient Modern UI Design

📱 Fully Responsive Design (Mobile-Friendly)

🏆 Performance-Based Grading System

🔁 Restart Quiz Option

⚡ Smooth Animations & Interactive Effects

The UI is designed with a futuristic dark theme and animated background effects to create a professional and immersive experience. Advanced CSS features such as gradients, animations, transitions, backdrop blur (glass effect), and responsive media queries are used to enhance visual appeal.

The JavaScript logic efficiently handles question rendering, answer selection, timer functionality, score calculation, progress tracking, and final result generation. At the end of the quiz, users receive:

Final Score (Out of 15)

Accuracy Percentage

Performance Grade (Web Mastery, Elite Developer, Professional Level, or Keep Learning)

Total Time Taken

Questions Attempted

The grading system motivates learners by categorizing performance levels based on their percentage score.
